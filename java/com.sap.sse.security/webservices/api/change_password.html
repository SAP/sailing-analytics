SAP Sponsorships Engine Security Webservices API

URL: /api/restsecurity/change_password

Description:

The service updates a user's password. This requires that user to be the currently authenticated subject or to have administrator permissions.


Request Method: POST
Output format: An Empty response with status code 200 in case of success; error-indicating status codes otherwise
Mandatory parameters:
username
password
Example: curl -X POST -d "username=uhl&password=123" "http://127.0.0.1:8888/security/api/restsecurity/change_password"
Will provide a JSON document as explained above.
