SAP Sponsorships Engine Security Webservices API

URL: /api/restsecurity/login

Description:

The service authenticates a subject based on POST form parameter and creates a session. The session is represented by the JSESSIONID cookie value which can be used in subsequent requests to the same session with the same authenticated subject.


Request Method: POST
Output format: Redirect (302 Found) to /api/restsecurity/hello which delivers a JSON document of the following format
{"principal":"admin","authenticated":true,"remembered":false}
Mandatory parameters:
username
password
Example: curl -L -D /tmp/header -d "username=admin&password=admin" http://127.0.0.1:8888/security/api/restsecurity/login
Will provide a redirect and a JSON document as explained above.
