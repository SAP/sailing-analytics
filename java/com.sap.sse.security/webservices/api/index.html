SAP Sport Sponsorships Engine Security Webservices API - Overview

The SAP Sport Sponsorships Engine (SSE) and the applications built with it support security with various authentication and authorization facilities. This module offers REST services that can be used to log in and log out a subject, obtain a bearer token for a subject and manage security concerns. By default, the base URL for these services is "/security", so a full login URL, for example, could be http://sapsailing.com/security/api/restsecurity/login.

Webservice URL Description
/api/restsecurity/create_user Create a user
/api/restsecurity/change_password Change a user's password
/api/restsecurity/forgot_password Trigger the process of password recovery
/api/restsecurity/user Read, update or delete a user
/api/restsecurity/has_permission Check whether the subject has one or more permissions.
/api/restsecurity/role Create, read, update or delete a role
/api/restsecurity/usergroup Read, update or delete a user group
/api/restsecurity/access_token Obtain a bearer access token
/api/restsecurity/hello Test that a subject is authenticated
/api/restsecurity/remove_access_token Remove an existing access token, rendering it invalid for authenticating to the server
/api/restsecurity/ownership Change Object ownership
