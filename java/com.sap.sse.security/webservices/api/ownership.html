SAP Sponsorships Engine Security Webservices API

URL: /api/restsecurity/onwership

Description:

The service changes the ownership of object.


Change Ownership
Request Method: POST
Output format: a JSON document of the following format
{ "responseStatus": "true", "responseMessage": "Ownership changed successfully"}
Parameter (JSON Body):
objectId
Unique Object id.
objectType
Object type represents the type of object to change the ownership.
groupId
GroupId to assign given group to ownership of object object.
username
Username to search the user in order to attach it with given Object ownership.
Example: curl -X POST -d '{ "objectId":"427ebd42-ac69-4f74-a171-376d9bea9457", "objectType":"USER_GROUP", "groupId":"admin-tenant", "username":"admin" }' "http://admin:admin@127.0.0.1:8888/security/api/restsecurity/ownership"
Will provide a JSON document as explained above.
