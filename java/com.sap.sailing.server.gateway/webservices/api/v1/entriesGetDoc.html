SAP Sailing Analytics Webservices API Version 1.0 Beta

URL: /api/v1/regattas/{name}/entries

Description:

The service gets the entries (competitor list) of a regatta


Webservice Type: REST
Output format: Json
Mandatory parameters: None
Optional parameters: None
Request method: GET
Example: http://www.sapsailing.com/sailingserver/api/v1/regattas/Kieler Woche (5o5)/entries
