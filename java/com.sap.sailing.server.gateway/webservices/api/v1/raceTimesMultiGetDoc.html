SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/regattas/{regattaname}/races/times

Description:

Gets the timing data for multiple races, including the start/end times, the tracking start/end times as well as the "split times" or "mark passing times" of the first competitor rounding each mark.
All slashes contained within the {regattaname} parameter must be replaced by "__" (double underscore).


Webservice Type: REST
Output format: Json
Mandatory parameters: None
Optional parameters: racename: One or more race names to retrieve the times race for
Request method: GET
Example: http://www.sapsailing.com/sailingserver/api/v1/regattas/TW%202013%20(Finn)/races/times
