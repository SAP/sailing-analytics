SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/leaderboard/{name}/update

Description:

The service can modify some of the details of a leaderboard. Currently this allows to update the discarding thresholds of a leaderboard.


Webservice Type: REST
Output format: Json
Mandatory parameters: None
Optional parameters: None
Request method: POST
Payload: *(optional) resultDiscardingThresholds: if parameter is absent or value is null, the discarding thresholds will not change. If passing an empty array[] the discarding thresholds will be cleared.
Example: http://www.sapsailing.com/sailingserver/api/v1/leaderboards/505 Worlds 2013/update
Request: 
		{
		  "resultDiscardingThresholds":[1,2]
		}
Response: Status 200
