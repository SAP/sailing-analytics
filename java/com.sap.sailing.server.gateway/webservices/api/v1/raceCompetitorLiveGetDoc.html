SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/regattas/{regattaname}/races/{racename}/competitors/live

Description:

Gets some competitor information (e.g current rank, current leg) for a live race


Webservice Type: REST
Output format: Json
Mandatory parameters: None
Optional parameters:
topN
Request method: GET
Example: http://www.sapsailing.com/sailingserver/api/v1/regattas/TW%202013%20(Finn)/races/Finn%20Race%204/competitors/live
