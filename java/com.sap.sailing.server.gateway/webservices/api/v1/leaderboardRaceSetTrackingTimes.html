SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/leaderboards/{name}/settrackingtimes?race_column={name}&fleet={name}

Description:

Modifies the tracking start/finish times for a race. If nothing is specified about the start of tracking time (none of the parameters startoftracking and startoftrackingasmillis present), no modification will be applied to the tracking start time, and no race log entry will be created for it. If one of the two parameters is provided but with an empty value, the tracking start time will be set to null explicitly.

Likewise, the same holds for endoftracking and endoftrackingmillis, accordingly.

The resulting JSON document tells which of the startoftracking and endoftracking times have been set for the race and to which value.


Webservice Type: REST
Output format: Json
Mandatory parameters: race_column, fleet
Optional parameters:
startoftracking an ISO date format to set the start of tracking; if empty and no startoftrackingasmillis is provided, the value will be set to null
startoftrackingasmillis alternative to startoftracking; provides the tracking start time as Unix timestamp in milliseconds since the epoch (1970-01-01T00:00:00Z)
endoftracking an ISO date format to set the end of tracking; if empty and no endoftrackingasmillis is provided, the value will be set to null
endoftrackingasmillis alternative to endoftracking; provides the tracking end time as Unix timestamp in milliseconds since the epoch (1970-01-01T00:00:00Z)
Request method: POST
Example: http://127.0.0.1:8888/sailingserver/api/v1/leaderboards/TracTrac%20Virtual%20Course%20Marks%20Test/settrackingtimes?race_column=R1&fleet=Default&startoftracking=2007-01-03T13:05:00Z
delivers 
		{
			"startoftracking": 1167829500000
		}
