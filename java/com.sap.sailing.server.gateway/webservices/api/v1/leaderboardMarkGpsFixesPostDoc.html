SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/leaderboards/{name}/marks/{id}/gps_fixes

Description:

Adds an arbitrary number GPS-fixes that originated from a device with a unique identifier as fixes for the mark. This is done by adding the fixes to the GPSFixStore, and creating device-mappings in the RegattaLog attached to the Leaderboard.


Webservice Type: REST
Post data input format: JSON
Output format: HTTP 200
Mandatory parameters: None
Optional parameters: secret: in case the given secret is equal to the one stored in the regatta defined by {name}, further security checks will be skipped.
Request method: POST
Example: POST http://www.sapsailing.com/sailingserver/api/v1/leaderboards/{name}/marks/{id}/gps_fixes
