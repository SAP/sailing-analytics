SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/boatclasses

Description:

Gets information about a boat class or the list of available boat classes


Webservice Type: REST
Output format: Json
Mandatory parameters: None
Optional parameters: boatclassname
Request method: GET
