SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/markProperties

Description:

The service gets all mark properties filtered by tags.


Webservice Type: REST
Output format: Json
Mandatory parameters: tag: tag for filtering. Multiple tags are allowed.
Optional parameters: None
Request method: GET
Example: http://www.sapsailing.com/sailingserver/api/v1/markproperties?tag=test1&tag=test2
