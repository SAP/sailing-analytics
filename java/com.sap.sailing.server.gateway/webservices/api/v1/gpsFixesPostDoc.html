SAP Sailing Analytics Webservices API Version 1.0 Beta

URL: /api/v1/gps_fixes

Description:

Adds an arbitrary number GPS-fixes that originated from a device with a unique identifier to the GPSFixStore. These can be mapped to a competitor or a mark via the DeviceMappings in the RaceLog-Tracking connector.


Webservice Type: REST
Post data input format: JSON
Output format: HTTP 200
Mandatory parameters: None
Optional parameters: None
Request method: POST
Example: POST the following JSON data to http://www.sapsailing.com/sailingserver/api/v1/gps_fixes 

{
  "deviceUuid" : "af855a56-9726-4a9c-a77e-da955bd289bf",
  "fixes" : [
    {
      "timestamp" : 14144160080000,
      "latitude" : 54.325246,
      "longitude" : 10.148556,
      "speed" : 3.61,
      "course" : 258.11,
    },
    {
      "timestamp" : 14144168490000,
      "latitude" : 55.12456,
      "longitude" : 8.03456,
      "speed" : 5.1,
      "course" : 14.2,
    }
  ]
}
