SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/regattas/{regattaName}/tracking_devices

Description:

The service gets the status of all GPS tracking devices associated to the regatta. The user needs to have UPDATE permission granted to be allowed to access tracking device data.


Webservice Type: REST
Output format: Json
Mandatory parameters: None
Optional parameters: None
Request method: GET
Example: http://www.sapsailing.com/sailingserver/api/v1/regattas/Kornati Cup 2019 - SUP Contest/tracking_devices
