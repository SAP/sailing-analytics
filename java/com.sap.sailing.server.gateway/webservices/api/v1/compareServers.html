SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/compareservers

Description:

Compares two SAP Sailing Analytics servers based on the content. The return contains either the missing leaderboardgroup name or if the leaderboardgroups match on both servers the missing parts within the leaderboardgroups. If the servers contain the exact same content the return will just have the given server URLs in it.


Webservice Type: POST
Output format: json
Mandatory parameters:
server1, the URL to the first server.
server2, the URL to the second server.
Request method: POST
Example: curl 127.0.0.1:8889/sailingserver/api/v1/compareservers -d "server1=127.0.0.1:8888" -d "server2=127.0.0.1:8889"
produces output:
        {
          "127.0.0.1:8889": [
            {
              "name": "EliminationTestEventLeaderboardGroup1"
            },
            {
              "name": "EliminationTestEventLeaderboardGroup2"
            }
          ],
          "127.0.0.1:8888": []
        }
