SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/compareservers

Description:

Compares two SAP Sailing Analytics servers based on the content. The return contains either the missing leaderboardgroup name or if the leaderboardgroupnames match on both servers the missing parts within the leaderboardgroups. If the servers contain the exact same content the return will just have the given server hostnames in it.


Webservice Type: GET
Output format: json
Mandatory query parameters:
server1, the hostname for the first server.
server2, the hostname for the second server.
Request method: GET
Example: curl localhost:8888/sailingserver/api/v1/compareservers?server1=localhost:8888&server2=localhost:8889
