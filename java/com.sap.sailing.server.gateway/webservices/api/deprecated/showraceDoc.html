SAP Sailing Analytics - Web Services

Name: showrace

Description:

This service exports race and competitor data, either for a specific time point or for the end of the race. Time points in the document are Unix millisecond time stamps.


Webservice Type: REST
Output format: Json
Base-Url: http://www.sapsailing.com/sailingserver/moderator?action=showrace
Mandatory parameters:
regattaname
racename
Optional parameters:
time
timeasmillis
sinceupdate
Request method: GET
Example: http://www.sapsailing.com/sailingserver/moderator?action=showrace&regattaname=ESS%202012%20Cardiff%20%28Extreme40%29&racename=Cardiff%20Race12
