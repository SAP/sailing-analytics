SAP Sailing Analytics - Web Services

Name: wind

Description:

This service lists GPS the fixes available for a race's competitors. A time range can be provided using optional parameters. It those are not used, all available GPS fixes are returned. Fixes classified as outlier (obviously wrong fixes due to tracker or transmission failure) will not show in the result.


Webservice Type: REST
Output format: Json
Base-Url: http://www.sapsailing.com/sailingserver/moderator?action=showboatpositions
Mandatory parameters:
regattaname
racename
Optional parameters:
since
sinceasmillis
to
toasmillis
Request method: GET
Example: http://www.sapsailing.com/sailingserver/moderator?action=showboatpositions&regattaname=ESS%202012%20Cardiff%20%28Extreme40%29&racename=Cardiff%20Race12
