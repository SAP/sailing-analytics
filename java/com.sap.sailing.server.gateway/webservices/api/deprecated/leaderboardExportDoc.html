SAP Sailing Analytics - Web Services

Name: leaderboard

Description:

This service exports the data of a leaderboard with all races, including total points, points per race and total rank.


Webservice Type: REST
Output format: Json
Base-Url: http://www.sapsailing.com/sailingserver/leaderboard
Mandatory parameters: leaderboardName
Optional parameters: resultState [Live = Default, Preliminary, Final]
  maxCompetitorsCount
Request method: GET
Example: http://www.sapsailing.com/sailingserver/leaderboard?leaderboardName=Extreme%20Sailing%20Series%202013%20Overall&resultState=Final
